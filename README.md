The project focuses on:
- Intelligent contact search

- Search optimization

- Real-time filtering

# Installation

## Clone Repository

```bash
git clone https://github.com/Chaitra905/Smart-PhoneBook.git
```

---

# Backend Setup

```bash
cd backend
npm install
```

Create `.env`

```env
PORT=5000
MONGO_URI=YOUR_MONGODB_URI
JWT_SECRET=YOUR_SECRET_KEY
```

Run Backend

```bash
npm run dev
```

---

# Frontend Setup

```bash
cd backend/frontend
npm install
```

Run Frontend

```bash
npm start
```
